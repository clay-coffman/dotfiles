# Window Management Shortcuts (KDE Plasma / KWin)

> **Modifier scheme:** Same vim keys (H/J/K/L) everywhere; the modifier indicates the action level.

| Modifier       | Action            | Mnemonic          |
| -------------- | ----------------- | ----------------- |
| `Meta`         | Window focus      | "Which window?"   |
| `Meta+Alt`     | Grid navigation   | "Where am I?"     |
| `Meta+Shift`   | Move window       | "Take it with me" |
| `Meta+Ctrl`    | Tile/resize       | "Shape it"        |

---

## Tier 1 — Window Focus (`Meta`)

| Shortcut | Action             |
| -------- | ------------------ |
| `Meta+H` | Focus window left  |
| `Meta+J` | Focus window down  |
| `Meta+K` | Focus window up    |
| `Meta+L` | Focus window right |

## Tier 2 — Grid Navigation (`Meta+Alt`)

Activities = rows (up/down), Desktops = columns (left/right).

| Shortcut     | Action               |
| ------------ | -------------------- |
| `Meta+Alt+H` | Desktop left         |
| `Meta+Alt+L` | Desktop right        |
| `Meta+Alt+J` | Next activity (down) |
| `Meta+Alt+K` | Prev activity (up)   |
| `Alt+1/2/3`  | Jump to activity     |

## Tier 3 — Move Window (`Meta+Shift`)

| Shortcut          | Action                        |
| ----------------- | ----------------------------- |
| `Meta+Shift+H`    | Move window to desktop left   |
| `Meta+Shift+L`    | Move window to desktop right  |
| `Meta+Shift+N`    | Move window to next screen    |
| `Meta+Shift+Left` | Move window to prev screen    |

## Tier 4 — Tile/Resize (`Meta+Ctrl`)

| Shortcut           | Action           |
| ------------------ | ---------------- |
| `Meta+Ctrl+H`      | Tile left half   |
| `Meta+Ctrl+J`      | Tile bottom half |
| `Meta+Ctrl+K`      | Tile top half    |
| `Meta+Ctrl+L`      | Tile right half  |
| `Meta+Ctrl+Return` | Toggle maximize  |
| `Meta+T`           | Tile editor      |

---

## Window Switching

| Shortcut                      | Action                             |
| ----------------------------- | ---------------------------------- |
| `Alt+Tab`; `Meta+Tab`        | Cycle windows                      |
| `Alt+Shift+Tab`; `Meta+Shift+Tab` | Cycle windows (reverse)       |
| `Meta+`` `` `; `Alt+`` `` ` | Cycle windows (current app)        |
| `Meta+~`; `Alt+~`            | Cycle windows (current app, rev.)  |

## Overview / Expose

| Shortcut   | Action                                |
| ---------- | ------------------------------------- |
| `Meta+W`   | Overview                              |
| `Meta+G`   | Grid View                             |
| `Ctrl+F9`  | Present Windows (current desktop)     |
| `Ctrl+F10` | Present Windows (all desktops)        |
| `Ctrl+F7`  | Present Windows (current app class)   |

## Window Actions

| Shortcut             | Action               |
| -------------------- | -------------------- |
| `Meta+D`             | Show Desktop         |
| `Alt+F3`             | Window Operations    |
| `Alt+F4`             | Close window         |
| `Meta+Ctrl+Esc`      | Kill window          |
| `Meta+Shift+Return`  | Maximize (alt)       |

## Activities

| Shortcut       | Action                   |
| -------------- | ------------------------ |
| `Meta+Q`       | Activity Switcher        |
| `Meta+A`       | Next activity (alt)      |
| `Meta+Shift+A` | Previous activity (alt)  |
| `Ctrl+Alt+Tab` | Switch to next activity  |

## Task Manager (pinned apps)

`Meta+1` through `Meta+9`

## Utilities

| Shortcut       | Action            |
| -------------- | ----------------- |
| `Meta+V`       | Clipboard popup   |
| `Meta+Ctrl+X`  | Clipboard actions |
| `Ctrl+Alt+L`   | Lock session      |
| `Ctrl+Alt+Del` | Log out screen    |
| `Meta+?`       | This cheatsheet   |
| `Meta+Shift+T` | Toggle theme      |

## Launcher

`Meta`; `Alt+F1`
